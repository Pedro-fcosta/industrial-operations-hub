# 6. MOVIMENTAÇÃO DE POSIÇÃO / LOTE
# 6.1. Criar rota "/rastreabilidade/movimentar".
# 6.2. Se o método for GET:
#      - Mostrar formulário de busca.
#      - Pedir CR e Posição.
# 6.3. Se o método for POST:
#      - Identificar qual ação foi enviada pelo formulário.
# 6.4. Se a ação for "buscar":
#      - Receber CR e Posição.
#      - Buscar na tabela "posicoes" todos os lotes com o mesmo CR e mesma Posição.
#      - Se nenhum lote for encontrado:
#           Exibir mensagem de erro.
#      - Se apenas um lote for encontrado:
#           Selecionar o lote automaticamente.
#           Mostrar o formulário de movimentação.
#      - Se mais de um lote for encontrado:
#           Mostrar uma tabela com todos os lotes encontrados.
#           A tabela deve mostrar:
#               Quantidade
#               Local atual
#               Última movimentação
#               Botão "Movimentar este lote"
# 6.5. Se a ação for "selecionar_lote":
#      - Receber o ID interno do lote.
#      - Buscar o lote na tabela "posicoes" usando o ID.
#      - Mostrar o formulário de movimentação desse lote.
# 6.6. Se a ação for "registrar":
#      - Receber ID interno do lote.
#      - Receber destino.
#      - Receber responsável.
#      - Receber observação.
#      - Se destino for "Outro":
#           Receber a descrição digitada.
#           Salvar destino como "Outro - descrição digitada".
#      - Buscar o lote na tabela "posicoes" pelo ID.
#      - Pegar o local atual do lote e usar como origem.
#      - Gerar data e hora atual.
#      - Inserir registro na tabela "movimentacoes".
#      - Salvar:
#           posicao_id
#           CR
#           Posição
#           Origem
#           Destino
#           Responsável
#           Observação
#           Data e hora
#      - Atualizar o registro da tabela "posicoes".
#      - Atualizar:
#           local_atual
#           responsavel
#           observacao
#           ultima_movimentacao
#      - Exibir mensagem de sucesso.
#      - Redirecionar para a tela de movimentação.
